# Emotion Recognition from Social Media Videos

## Overview

This project was developed as part of **Satria Data 2025**, one of Indonesia's national data science competitions. The objective was to build a multimodal emotion recognition system capable of identifying emotions expressed in Indonesian social media videos.

The project integrates information from video, audio, and textual modalities to better understand emotional expressions in user-generated content from platforms such as TikTok, Instagram, and Facebook.

## My Role

**Team Leader & Machine Learning Developer**

As the team leader, I coordinated project planning, task allocation, dataset preparation, and model development activities. I was also directly involved in data validation, annotation review, duplicate detection, and multimodal feature engineering.

## Problem Statement

Social media videos contain rich emotional information that cannot be fully captured through a single modality. Understanding emotions from video content requires combining visual, audio, and textual signals while ensuring data quality and annotation consistency.

## Dataset Preparation

Key activities included:

* Collecting Indonesian-language social media videos from multiple platforms.
* Converting speech into text transcripts.
* Detecting duplicate, reposted, clipped, and extended videos using semantic similarity analysis.
* Reviewing annotations and validating label consistency.
* Performing data cleaning and quality assurance procedures.
